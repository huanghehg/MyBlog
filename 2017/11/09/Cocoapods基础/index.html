一、命令行安装

1、升级ruby环境
sudo gem update --system
Password:/*此处输入密码*/

当系统提示 RubyGems system software updated 说明更新成功

2、ruby环境更新完成就可以安装cocoa pods
gem sources --remove https://rubygems.org/
gem sources -a https://ruby.taobao.org/

以上两步是为了替换sources路径（天朝不解释）

gem sources -l
*** CURRENT SOURCES ***
https://ruby.taobao.org/

替换成功执行

sudo gem install cocoapods

输入密码之后就坐等成功

二、使用xcode安装

直接去github下载工程，运行，重启xcode，搞定

安装Alcatraz 插件

curl -fsSL https://raw.githubusercontent.com/supermarin/Alcatraz/deploy/Scripts/install.sh | sh

重启xcode 打开Window

选择Package Manager

输入要安装的插件，install就ok

三、cocoapods的简单使用 （仅限命令行）

1、首先 cd到项目文件中
2、新建名字为Podfile的文件,项目中有Podfile文件的直接走到 4
3、打开 Podfile ，输入 platform:ios, ‘7.0‘
4、在终端输入pod search sdwebimage

5、找到你所需要的第三方库 找到 pod 'SDWebImage', '~> 3.7.5'，复制到Podfile文件中，其中3.7.5是版本号 ，可以修改为 versions中的任意一个

6、保存Podfile 在终端执行 pod instal 等待一会儿第三方库会自动clone在项目中（该过程时间有点久 可以在后面加上参数 --verbose --no-repo-update ，不更新repo）

注意：对于项目中有Podfile文件，一定要cd到Podfile 文件所在的目录 执行pod install
上传到git时，只需要上传Podfile 文件，其他自动生成的文件不要上传
