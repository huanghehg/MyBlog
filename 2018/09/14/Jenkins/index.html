安装

1、使用brew install jenkins

启动: brew services start jenkins

停止: brew services stop jenkins

重启: brew services restart jenkins

  • 改端口
    • which jenkins
    • 打开路径






2、官网下包安装不推荐感觉不爽

启动: sudo launchctl unload /Library/LaunchDaemons/org.jenkins-ci.plist

停止: sudo launchctl load /Library/LaunchDaemons/org.jenkins-ci.plist

改端口: sudo defaults write /Library/Preferences/org.jenkins-ci httpPort 1234
